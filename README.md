# Shoppie - Online Store 

A modern, responsive e-commerce storefront built with **HTML, CSS, and JavaScript**, featuring a real-time backend powered by **Firebase**.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend as a Service (BaaS):** Firebase (Authentication & Realtime Database)
* **Hosting:** Firebase Hosting
